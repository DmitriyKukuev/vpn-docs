# Подключение к AmneziaVPN

## 1. Скачивание приложения

Выберите вашу платформу:

- [Android — Google Play](https://play.google.com/store/apps/details?id=org.amnezia.vpn&utm_source=amnezia.org&utm_campaign=organic&utm_medium=referral)
- [Windows — GitHub Releases](https://github.com/amnezia-vpn/amnezia-client/releases/download/4.8.14.5/AmneziaVPN_4.8.14.5_x64.exe)
- [macOS — GitHub Releases](https://github.com/amnezia-vpn/amnezia-client/releases/download/4.8.14.5/AmneziaVPN_4.8.14.5_macos.pkg)
- [iOS — App Store](https://apps.apple.com/ru/app/amneziawg/id6478942365)

## 2. Файл с настройками

Файл конфигурации выдаёт оператор сервиса. Без него подключиться не получится.

## 3. Подключение в приложении

### AmneziaVPN (Android, Windows, macOS) {#android-windows-macos}

1. Нажмите «Файл с настройками подключения» («File with connection settings») и выберите файл на вашем устройстве

   ![Файл с настройками подключения](/images/amneziavpn/amneziavpn-step-1.webp)

2. Нажмите «Подключиться» («Connect») на экране импорта

   ![Подключиться](/images/amneziavpn/amneziavpn-step-2.webp)

3. Нажмите кнопку «Подключиться» («Connect») на главном экране

   ![Главный экран](/images/amneziavpn/amneziavpn-step-3.webp)

### AmneziaWG (iOS) {#ios}

1. Нажмите «Добавить туннель»

   ![Добавить туннель](/images/amneziavpn/amneziawg-step-1.webp)

2. Нажмите «Создать из файла или архива» и выберите файл на вашем устройстве

   ![Создать из файла](/images/amneziavpn/amneziawg-step-2.webp)

3. Нажмите «Разрешить»

   ![Разрешить](/images/amneziavpn/amneziawg-step-3.webp)

4. Включите переключатель рядом с названием конфигурации

   ![Включить туннель](/images/amneziavpn/amneziawg-step-4.webp)
