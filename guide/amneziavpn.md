# Подключение к AmneziaVPN через файл конфигурации

## 1. Скачать приложение

Выберите вашу платформу:

- [Android — Google Play](https://play.google.com/store/apps/details?id=org.amnezia.vpn&utm_source=amnezia.org&utm_campaign=organic&utm_medium=referral)
- [Windows — GitHub Releases](https://github.com/amnezia-vpn/amnezia-client/releases/download/4.8.14.5/AmneziaVPN_4.8.14.5_x64.exe)
- [macOS — GitHub Releases](https://github.com/amnezia-vpn/amnezia-client/releases/download/4.8.14.5/AmneziaVPN_4.8.14.5_macos.pkg)
- [iOS — App Store](https://apps.apple.com/ru/app/amneziawg/id6478942365)

## 2. Скачать файл конфигурации

Файл конфигурации выдаёт оператор сервиса. Обычно это файл с расширением `.conf`. Сначала скачайте его на устройство, с которого будете подключаться. Без этого файла импорт конфигурации не получится.

## 3. Подключиться в приложении

### AmneziaVPN (Android, Windows, macOS) {#android-windows-macos}

1. Откройте приложение AmneziaVPN. Если это первый запуск, нажмите «Приступим».

2. На экране добавления подключения нажмите «Файл с настройками подключения».

3. В проводнике или приложении «Файлы» выберите полученный `.conf`-файл.

4. На экране нового подключения проверьте, что выбран нужный файл, и нажмите «Подключиться».

   ![Импорт файла конфигурации в AmneziaVPN](/images/amneziavpn-file/amneziavpn-1.webp)

5. После импорта на главном экране еще раз нажмите большую кнопку «Подключиться».

   ![Подключение в AmneziaVPN](/images/amneziavpn-file/amneziavpn-2.webp)

### AmneziaWG (iOS) {#ios}

1. Откройте приложение AmneziaWG и нажмите «Добавить туннель».

2. Выберите «Создать из файла или архива» и укажите скачанный файл конфигурации.

3. Когда iPhone попросит разрешение на добавление VPN-конфигурации, нажмите «Разрешить».

4. После импорта включите переключатель рядом с названием вашей конфигурации.

   ![Импорт файла конфигурации в AmneziaWG](/images/amneziavpn-file/amneziawg.webp)
