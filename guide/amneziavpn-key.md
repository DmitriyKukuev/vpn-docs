# Подключение к AmneziaVPN через ключ в виде текста

## 1. Скачать приложение

Выберите вашу платформу:

- [Android — Google Play](https://play.google.com/store/apps/details?id=org.amnezia.vpn)
- [Windows — GitHub Releases](https://github.com/amnezia-vpn/amnezia-client/releases/download/4.8.19.0/AmneziaVPN_4.8.19.0_x64.exe)
- [macOS — GitHub Releases](https://github.com/amnezia-vpn/amnezia-client/releases/download/4.8.19.0/AmneziaVPN_4.8.19.0_macos.pkg)
- [iOS — App Store](https://apps.apple.com/ru/app/defaultvpn/id6744725017)

## 2. Скопировать ключ

Текстовый ключ выдаёт оператор сервиса. Обычно он начинается с `vpn://`.

Сам ключ нужно предварительно скопировать целиком на подключаемом устройстве. Если ключ скопирован не полностью или с лишними символами, приложение сообщит об ошибке.

## 3. Подключиться в приложении

### AmneziaVPN (Android, Windows, macOS) {#android-windows-macos}

1. Откройте приложение AmneziaVPN и нажмите на иконку `➕` (плюс) или на кнопку `Приступим`, если у вас не было других подключений.

2. Нажмите кнопку `Вставить` возле поля для ввода ключа → `Продолжить` → `Подключиться`.

   ![Импорт ключа в AmneziaVPN](/images/amneziavpn-key/amneziavpn-1.webp)

Создастся новое подключение — можно подключаться.

   ![Вставка ключа в AmneziaVPN](/images/amneziavpn-key/amneziavpn-2.webp)

### DefaultVPN (iOS) {#ios}

1. Скачайте и откройте приложение `DefaultVPN`.

2. Нажмите на иконку `➕` (плюс) → `Вставить` ниже поля для ввода ключа → `Добавить`.

3. Создастся новое подключение — можно подключаться.

   ![Импорт ключа в DefaultVPN](/images/amneziavpn-key/defaultvpn.webp)
