# Подключение к Xray

Используем протокол **VLESS** с маскировкой **Reality** и транспортом **XHTTP**.

## 1. Скачать приложение

Рекомендуем использовать Happ. Это приложение для подключения к Xray на разных платформах.

Выберите вашу платформу:

- [iOS — App Store](https://apps.apple.com/ru/app/happ-proxy-utility-plus/id6746188973)
- [Android — Google Play](https://play.google.com/store/apps/details?id=com.happproxy)
- [Windows — GitHub Releases](https://github.com/Happ-proxy/happ-desktop/releases/latest/download/setup-Happ.x64.exe)
- [macOS — GitHub Releases](https://github.com/Happ-proxy/happ-desktop/releases/latest/download/Happ.macOS.universal.dmg)
- [Другие платформы — сайт Happ](https://www.happ.su/main/ru)

Также можно использовать другие приложения с поддержкой Xray. Список клиентов доступен в разделе [GUI Clients](https://github.com/xtls/xray-core#gui-clients).

## 2. Получить ссылку подписки

Ссылку подписки выдаёт оператор сервиса. Она понадобится для перехода в браузере или для вставки в приложение.

## 3. Подключиться в приложении

Выберите удобный способ подключения.

### Способ 1. Вставить ссылку подписки в Happ

#### Android и iOS

1. Скопируйте ссылку подписки.

2. Откройте Happ и нажмите `Из буфера` внизу экрана.

   Также можно нажать `+` в правом верхнем углу и выбрать `Вставить из буфера обмена`.

   ![Добавление ссылки подписки в Happ](/images/xray/xray-link-1.webp)

3. Подключение добавится в приложение. Нажмите большую кнопку включения VPN.

   ![Подключение в Happ](/images/xray/xray-link-2.webp)

#### Windows и macOS

1. Скопируйте ссылку подписки.

2. Откройте Happ и нажмите `Из буфера`.

   ![Добавление ссылки подписки из буфера в Happ на компьютере](/images/xray/xray-link-pc-1.webp)

3. Также можно нажать `+` слева сверху, выбрать добавление подписки, вставить ссылку в поле `URL подписки` и внизу нажать `Добавить`.

   ![Добавление ссылки подписки вручную в Happ на компьютере](/images/xray/xray-link-pc-2.webp)

4. Подключение добавится в приложение. Нажмите большую кнопку включения VPN.

   ![Подключение в Happ на компьютере](/images/xray/xray-link-pc-3.webp)

### Способ 2. Добавить подключение со страницы подписки

1. Откройте ссылку подписки в браузере.

2. Внизу страницы выберите вашу платформу: `Android` или `iOS`.

3. В выпадающем списке выберите `Happ`.

   ![Выбор Happ для iOS](/images/xray/xray-deeplink-1.webp)

   ![Выбор Happ для Android](/images/xray/xray-deeplink-2.webp)

4. Подтвердите добавление подключения в Happ.

### Способ 3. Подключиться по QR-коду

1. Откройте Happ.

2. Нажмите `QR-Код` внизу экрана.

   Также можно нажать `+` в правом верхнем углу и выбрать `QR-код`.

   ![Сканирование QR-кода в Happ](/images/xray/xray-qr-1.webp)

3. На странице подписки отсканируйте QR-код в блоке `Подписка`.

   ![QR-код подписки](/images/xray/xray-qr-2.webp)

4. Подтвердите добавление подключения в Happ.
